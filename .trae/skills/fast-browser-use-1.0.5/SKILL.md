# 浏览器自动化技能

## 功能概述

本技能提供全面的浏览器自动化能力，包括：

- **网页访问**：打开URL、导航、刷新
- **数据获取**：提取文本、图片、链接、元素
- **页面操作**：点击、输入、滚动、截图
- **会话管理**：cookies、local storage、多标签
- **性能分析**：页面加载时间、DOM提取

## 技术实现

- **Rust核心**：高性能浏览器自动化引擎
- **Chrome CDP**：直接控制Chrome浏览器
- **多平台支持**：Windows、macOS、Linux

## 使用场景

- **数据采集**：从网站提取结构化数据
- **网页测试**：自动化UI测试、表单填写
- **信息检索**：快速获取网页内容
- **屏幕截图**：生成网页截图
- **会话管理**：保持登录状态进行操作

## 调用方式

当检测到浏览器相关操作时，系统会自动调用本技能。也可以通过以下方式手动调用：

```
使用浏览器技能访问：[URL]
提取网页内容：[URL]
网页截图：[URL]
```

## 性能优势

- **启动速度**：< 50ms
- **内存占用**：15 MB
- **DOM提取**：零拷贝技术

## 安全考虑

- **权限控制**：仅处理用户授权的操作
- **资源限制**：防止过度消耗系统资源
- **错误处理**：健壮的错误处理机制
